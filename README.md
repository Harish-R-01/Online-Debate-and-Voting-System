# 🗳️ Online Debate and Voting System

A web-based platform that enables users to create, participate in, and vote on debates. Built using HTML, CSS, JavaScript, Firebase Authentication, Cloud Firestore, and Firebase Storage.

## Features

- User Authentication (Sign Up, Login, Logout)
- Create and Manage Debates
- View Personal and Public Debates
- Real-Time Voting System
- Responsive Design for Desktop and Mobile
- Secure Data Storage with Firebase

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Firebase Authentication
- Cloud Firestore
- Firebase Storage
